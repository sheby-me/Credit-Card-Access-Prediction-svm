# Credit-Card-Access-Prediction-svm
Support Vector Machine model predicting whether a bank credit card will be accessed or not.

This Machine Learning project predicts whether a customer will access a bank credit card using Support Vector Machine (SVM).

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## Algorithm
Support Vector Machine (SVM)

## Features
- Data preprocessing
- Model training
- Accuracy evaluation
- Confusion matrix visualization
